University final project / dissertation

Researched, Planned and implemented a prototype iOS application in XCode and Swift. It was developed integrating Google’s Firestore NoSQL database to store any user generated data and also FSCalendar through CocoaPods. The end goal of the project was to create a to-do list application synched across all the users’ devices where tasks are assigned to users with a set deadline and they are able to report the progress of it. The second prototype of the application is currently in the planning stage and aims to integrate a role-base access control and further implement the calendar view for future planning.
